# Fun Calculator 🎉

Welcome to the **Fun Calculator** — a simple and playful program that lets you add, subtract, multiply, and divide two numbers like a boss! 😎

---

## Features

- Supports decimal inputs (thanks to `float()`).
- Performs basic arithmetic operations:
  - Addition ➕
  - Subtraction ➖
  - Multiplication ✖️
  - Division ➗

---

## How to Use

1. Run the Python script.
2. When prompted, enter the **first number** (decimals allowed).
3. Enter the **second number**.
4. Watch the calculator work its magic and display:
    - Sum
    - Difference
    - Product
    - Quotient

---

## Important Notes

- Please **do not enter zero as the second number** to avoid division by zero errors.
- This is a simple example intended for fun and learning.

---

## Example

Enter the first number: 10.5
Enter the second number: 2

Results of your two numbers:
Sum: 12.5
Difference: 8.5
Product: 21.0
Quotient: 5.25
